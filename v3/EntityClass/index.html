1.15.4.1.41 v3 Code System EntityClass

This code system (http://hl7.org/fhir/v3/EntityClass) is defined as part of HL7 v3.

Formal value Set definition (identifier http://hl7.org/fhir/v3/vs/EntityClass): XML or JSON

Release Date: 2014-12-11

OID for code system: 2.16.840.1.113883.5.41

OID for value set: 2.16.840.1.113883.1.11.10882 (this is the value set that includes the entire code system)

1.15.4.1.41.1 Description

Classifies the Entity class and all of its subclasses. The terminology is hierarchical. At the top is this HL7-defined domain of high-level categories (such as represented by the Entity subclasses). Each of these terms must be harmonized and is specializable. The value sets beneath are drawn from multiple, frequently external, domains that reflect much more fine-grained typing.
