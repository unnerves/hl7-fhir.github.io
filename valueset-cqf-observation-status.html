1.15.2.1.690 Value Set http://hl7.org/fhir/vs/cqf-observation-status

This is a value set defined by the FHIR project.

Codes providing the status of a CQF observation

Detailed Descriptions: XML or JSON.

This value set contains 2 concepts

1.15.2.1.690.1 CQFObservationStatus

Codes providing the status of an observation

This value set defines its own terms in the system http://hl7.org/fhir/observation-status

Code Display Definition
final The observation is complete and verified by an authorized person.
amended The observation has been modified subsequent to being Final, and is complete and verified by an authorized person.

This value set is used in the following places:

  • Profile cqf-observation: Observation.status required
  • Profile cqf-observation-absent: Observation.status required

The OID for the value set is 2.16.840.1.113883.4.642.2.690 (OIDs are not used in FHIR, but may be used in v3, or OID based terminology systems).
