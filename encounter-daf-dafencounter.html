Profile: DAFEncounter

The official URL for this profile is: 

http://hl7.org/fhir/Profile/encounter-daf-dafencounter

. Defines constraints and extensions on the Encounter resource for use in querying and retrieving a patient's encounter related information. : U.S. Data Access Framework (DAF) Encounter profile

This profile was published on Thu, Aug 21, 2014 00:00+0000 as a draft by U.S. Office of the National Coordinator (ONC).

Summary

Terminology Bindings

  • Encounter.status must come from Current state of the encounter
  • Encounter.class must come from Classification of the encounter
  • Encounter.type should come from This value set defines a set of codes that can be used to indicate the type of encounter.
  • Encounter.reason must come from Value set to describe the specific encounter code

Required/Prohibited Elements

  • The element Encounter is required
  • The element Encounter.identifier is required
  • The element Encounter.status is required
  • The element Encounter.class is required
  • The element Encounter.subject is required
